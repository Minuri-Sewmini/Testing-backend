# 🚀 Backend Starter Project

A Node.js & Express backend integrated with MongoDB Atlas, following the MVC architecture and ES Modules syntax.

## 🛠 Tech Stack
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas (Mongoose)
- **Syntax:** ES Modules (import/export)
- **Containerization:** Docker
